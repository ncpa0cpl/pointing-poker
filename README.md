# Pointing Poker Web App

Check it out [in here]([pointingpoker.site](https://pointingpoker.site))

## Setup

Clone the repo

```sh
git clone git@github.com:ncpa0cpl/pointing-poker.git
```

Go into the repo directory

```sh
cd pointing-poker
```

Install dependencies

```sh
bun i
```

Build it

```sh
bun build:prod
```

Run it

```sh
bun start
```
